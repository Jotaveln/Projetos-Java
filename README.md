## Projetos

### Sistema Bancário

Aplicação para gerenciamento de contas bancárias com operações básicas.

**Funcionalidades:**

* Cadastro de clientes
* Criação de contas bancárias
* Depósito e saque
* Transferência entre contas
* Listagem de contas
* Controle de saldo

**Tecnologias:**

* Java

**Conceitos aplicados:**

* Programação Orientada a Objetos
* Estruturas de dados (List, ArrayList)
* Organização de código em classes

## Objetivo

Desenvolver habilidades em programação Java com foco em lógica, orientação a objetos e estruturação de sistemas.
