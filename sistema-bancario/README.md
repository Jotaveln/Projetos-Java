# Sistema Bancário em Java

Projeto desenvolvido para praticar programação orientada a objetos e simular o funcionamento básico de um sistema bancário.

## Sobre o projeto
Este sistema permite criar contas, realizar depósitos, saques e transferências entre contas. O objetivo é aplicar conceitos fundamentais de Java e desenvolver lógica de programação voltada para sistemas reais.

## Funcionalidades
- Criação de contas bancárias
- Depósito de valores
- Saque de valores
- Transferência entre contas
- Consulta de saldo

## Conceitos aplicados
- Programação orientada a objetos (POO)
- Criação de classes e métodos
- Encapsulamento
- Organização de código em pacotes
- Uso de listas (ArrayList)

## Tecnologias utilizadas
- Java
- IntelliJ IDEA
- Git e GitHub

## Objetivo do projeto
Desenvolver habilidades em Java e construir projetos práticos para compor portfólio profissional.
